官方代码维护

一般功能定制后，为避免出现问题，EduSoho主系统的更新就无法同时更新。但EduSoho不断的更新会让您的网校时刻保持最强的产品竞争力，为了保持与EduSoho主系统的升级同步，可以寻求EduSoho官方的代码维护升级服务，具体请联系官方人员。

 

APP定制

目前，每个EduSoho都拥有免费的移动学习客户端，但为了凸显网校品牌，很多教育机构想以低成本实现自有的网校，EduSoho提供定制app服务，详细请与EduSoho官方客服联系。
